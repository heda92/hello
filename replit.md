# Hello World Landing Page

## Overview
A visually stunning "Hello World" landing page built with Bun. Features animated floating shapes, confetti effects, and random Catppuccin color palettes with designer jokes.

## Project Architecture
- **Runtime**: Bun
- **Server**: `server.js` - Simple Bun HTTP server on port 5000
- **Frontend**: `index.html` - Single-page static HTML with inline CSS/JS
- No external dependencies required

## How to Run
```
bun server.js
```
